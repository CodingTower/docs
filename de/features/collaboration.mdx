Mit ZenOpus können Sie gemeinsam Websites erstellen. Laden Sie Ihren Designer, Entwickler, Ihre Agentur oder andere Personen ein. Jeder sieht Veränderungen, wenn sie passieren.

Es gibt zwei Möglichkeiten der Zusammenarbeit. Sie können Mitarbeiter zu einem bestimmten Projekt oder zu Ihrem gesamten Workspace einladen.

## Mitarbeit an einem konkreten Projekt

Sie können Benutzer zur Zusammenarbeit an einem bestimmten Projekt einladen, indem Sie im Projekteditor auf „Einladen“ klicken. Fügen Sie ihre E-Mail-Adresse hinzu und sie erhalten eine Einladung per E-Mail.

<Note>
Mitwirkende nutzen Credits aus dem Workspace des Projekteigentümers.
</Note>

## Mitarbeit in einem Workspace

### Was ist ein Workspace?

Ein Workspace ist Ihr gemeinsamer Raum zum Erstellen und Zusammenarbeiten an Projekten. Sie können Ihre Teamkollegen zu Ihrem Workspace einladen und sie haben Zugriff auf alle darin enthaltenen Projekte. Sie können die Projekte der Arbeitsbereiche sehen, bearbeiten und veröffentlichen.

<Warning>
Für eine detaillierte Berechtigungskontrolle und Rollenzuweisung in Ihrem Workspace ist ein kostenpflichtiges Abonnement erforderlich.
</Warning>

Wenn Sie einen Plan abonnieren, wird dieser an Ihren gewünschten Workspace gebunden und dann nutzen alle Projekte im Workspace diese Credits, unabhängig davon, wer die Nachrichten sendet.

Sie können Ihren Workspace in den Workspace-Einstellungen anpassen, indem Sie:

- Umbenennen
- Den Avatar ändern
- Eine Beschreibung schreiben

### Wie füge ich Mitarbeiter in einem Workspace hinzu und verwalte sie?

Sie verwalten und fügen Mitarbeiter über die Registerkarte „Personen“ in den Workspace-Einstellungen hinzu. Sie können darauf zugreifen, indem Sie zu **Einstellungen** > **Personen** gehen.

Hier können Sie:

- Fügen Sie neue Mitarbeiter hinzu, indem Sie deren E-Mail-Adresse eingeben. Sie erhalten per E-Mail eine Einladung, die sie dann annehmen können.
- Verwalten Sie bestehende Mitarbeiter, indem Sie ihre Rollen ändern oder sie entfernen.
- Sehen Sie sich die monatliche Credits-Nutzung für jeden Mitarbeiter an

## FAQ

<AccordionGroup>
<Accordion title="Wie viele Mitglieder kann ein Workspace haben?">
Workspaces Support unbegrenzte Mitglieder für alle Pläne. Für Arbeitsbereiche mit Enterprise-Plänen können optional benutzerdefinierte Mitgliedergrenzen festgelegt werden.
</Accordion>
<Accordion title="Kann ich mehrere Arbeitsbereiche haben?">
Ja, aber Sie benötigen für jedes einzelne Abonnement.
</Accordion>
<Accordion title="Kann ich zwischen Arbeitsbereichen wechseln?">
Ja! Gehen Sie zum Haupt-Dashboard. Tippen Sie dort oben links auf Ihren Workspace-Namen. Im Dropdown-Menü sehen Sie die anderen Arbeitsbereiche, auf die Sie Zugriff haben. Zum Wechseln klicken Sie einfach auf den gewünschten Workspace.
</Accordion>
<Accordion title="Ist es möglich, Projekte zwischen Arbeitsbereichen zu übertragen?">
Ja! Gehen Sie zu den Projekteinstellungen, wo Sie die Option zum **Übertragen** des Projekts finden.
</Accordion>
<Accordion title="Kann ich den Workspace-Besitzer ändern?">
Ja, ein Eigentümer kann mehrere Workspace-Eigentümer hinzufügen.
</Accordion>
<Accordion title="Kann ich den Projekteigentümer ändern?">
Im Moment ist dies nicht möglich, aber Sie können das Projekt jederzeit in einem neuen Konto remixen und werden dann Eigentümer dieser Kopie.
</Accordion>
</AccordionGroup>
