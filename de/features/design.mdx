Mit **visuellen Bearbeitungen** können Sie Ihr Projekt visuell anpassen und schnelle, gezielte Änderungen an Layouts, Texten, Farben und Bildern vornehmen, ohne Code schreiben oder Credits verwenden zu müssen.

## Warum visuelle Bearbeitungen verwenden?

Visuelle Bearbeitungen ermöglichen es allen – Designern, Vermarktern und nicht-technischen Teamkollegen –, Änderungen visuell vorzunehmen und schneller voranzukommen.

### Hauptvorteile

- **Schnellere Iteration:** Aktualisieren Sie Layouts und Text visuell – kein Code oder Eingabeaufforderungen erforderlich
- **Barrierefreie Bearbeitung:** Auch technisch nicht versierte Benutzer können problemlos Änderungen vornehmen
- **Live-Vorschau**: Sehen Sie jede Änderung in Echtzeit, bevor Sie sie veröffentlichen

### Häufige Anwendungsfälle

- Vermarkter starten schnell neue Kampagnenseiten
- Produktmanager verfeinern Layouts und kopieren direkt
- Teams erstellen schnell Prototypen und Varianten

## Schlüsselfunktionen

Im Modus **Visuelle Bearbeitung** können Sie Ihr Projekt direkt auf der Seite visuell ändern. Es ist für schnelle, intuitive Anpassungen mit vollständiger Kontrolle über Layout und Abstände konzipiert.

- **Elemente mehrfach auswählen** für Stapelbearbeitungen, indem Sie `⌘ Command` (Mac) oder `⊞ Win` (Windows) gedrückt halten.
- **Layout-Steuerelemente** für Ausrichtung und Positionierung
- **Ränder und Polsterung anpassen** auf einzelnen Seiten von Containern
- **Bearbeiten Sie Text, Farben und Schriftarten** direkt in der Seitenleiste
- **Verbesserte Textformatierung** mit Zeilenumbrüchen und Inline-Stil
- **Ersetzen Sie Bilder**, indem Sie Dateien hochladen oder Bild-URLs hinzufügen
- **Generieren Sie KI-Bilder**, indem Sie Bilder mit Eingabeaufforderungen bearbeiten

## Schnellstart

1. Öffnen Sie die **Entwurfsansicht** über die Schaltfläche `+` neben „Vorschau“.
1. Wählen Sie Elemente in der Live-Vorschau aus und bearbeiten Sie sie visuell über das linke Bedienfeld.
    <Tip>
    Für einen schnellen Zugriff verwenden Sie die Verknüpfung **Visuelle Bearbeitungen** im Eingabeaufforderungsfeld.
    </Tip>
1. **Vorschau ansehen und speichern**, wenn Sie mit dem Ergebnis zufrieden sind.

<Warning>
Für visuelle Bearbeitungen werden keine Credits abgezogen, außer wenn:

- Änderungen an dynamischen Elementen vornehmen.
- Aufforderung an den ZenOpus-Agenten, das Design zu ändern. Verwenden Sie stattdessen visuelle Bearbeitungen.
</Warning>
