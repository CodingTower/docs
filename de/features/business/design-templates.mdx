## Was sind Designvorlagen?

Designvorlagen sind für die Pläne **Business** und **Enterprise** verfügbar. Sie ermöglichen es Teams, genehmigte Designs und Projektstrukturen wiederzuverwenden, um projektübergreifend ein einheitliches Design, Branding und Benutzererlebnis aufrechtzuerhalten.

Eine Designvorlage ist ein ZenOpus-Projekt, das als wiederverwendbar markiert wurde. Wenn eine Vorlage verwendet wird, wird die gesamte Projektcodebasis in das neue Projekt kopiert, einschließlich:

- Dateistruktur
- Komponenten und Stile
- Konfigurationsdateien
- Projektaufbau und Gerüstbau

Aus Designvorlagen erstellte Projekte sind völlig unabhängig und können frei bearbeitet werden.

## Warum Designvorlagen verwenden?

Verwenden Sie Designvorlagen, um:

- Behalten Sie konsistente Designsysteme und Architektur bei
- Starten Sie neue Projekte mit bewährten Codebasen
- Reduzieren Sie sich wiederholende Einrichtungsarbeiten
- Best Practices Workspace teilen

Designvorlagen helfen Teams, schneller voranzukommen und gleichzeitig die Projekte aufeinander abzustimmen.

## Voraussetzungen

So verwenden Sie Designvorlagen:

- Sie müssen einen **Business- oder Enterprise-Plan** haben
- Das Vorlagenprojekt muss über **öffentliche oder Workspace-Sichtbarkeit** verfügen.
- Sie müssen Projektbearbeiter, Administrator oder Eigentümer sein, um Projekte als Vorlagen festzulegen
- Sie müssen ein Workspace-Administrator oder Besitzer sein, um Standardvorlagen verwalten zu können

## Richten Sie eine Designvorlage ein

Um eine Designvorlage einzurichten, müssen Sie Projektbearbeiter, Administrator oder Eigentümer sein.

1. Öffnen Sie ein vorhandenes Projekt.
1. Gehen Sie zu **Projekteinstellungen**.
1. Legen Sie die Sichtbarkeit des Projekts auf **öffentlich** oder **Workspace fest.**
1. Aktivieren Sie **Als Vorlage verwenden.**

Das Projekt erscheint im Abschnitt **Workspace-Einstellungen → Vorlagen** und kann beim Erstellen neuer Projekte verwendet werden.

## Starten Sie ein Projekt anhand einer Designvorlage

So verwenden Sie eine Vorlage beim Erstellen eines neuen Projekts:

1. Klicken Sie im Eingabeaufforderungsfeld auf `+`.
1. Gehen Sie zu **Design → Vorlage verwenden**.
1. Wählen Sie eine Vorlage und erstellen Sie Ihr Projekt wie gewohnt.

Die ausgewählte Vorlage wird als Projektgrundlage verwendet. Ihre Eingabeaufforderung steuert weiterhin, was generiert wird.

## Standardvorlagen

Jeder Workspace kann **eine Standard-Designvorlage** haben. Standardvorlagen helfen dabei, die Art und Weise zu standardisieren, wie Projekte in einem Workspace erstellt werden. Sie müssen ein Workspace-Administrator oder Besitzer sein, um Standardvorlagen verwalten zu können.

So legen Sie eine Standardvorlage fest:

1. Gehen Sie zu **Workspace-Einstellungen → Vorlagen**.
1. Bewegen Sie den Mauszeiger über eine Vorlage und wählen Sie **Als Standard festlegen** aus.

Wenn eine Standardvorlage festgelegt ist, wird diese im Eingabeaufforderungsfeld angezeigt und automatisch auf neue Projekte angewendet. Sie können die Auswahl jederzeit überschreiben.

So entfernen Sie eine Standardvorlage:

1. Gehen Sie zu **Workspace-Einstellungen → Vorlagen**.
1. Bewegen Sie den Mauszeiger über die Vorlage und entfernen Sie die Standardeinstellung.
