Sie können verschiedene Labs-Funktionen aktivieren und deaktivieren, indem Sie zu Ihren [Kontoeinstellungen] (https://zenopus.dev/settings/account) gehen und dann zu **Labs** navigieren.

<Note>
Diese experimentellen Funktionen können jederzeit geändert oder entfernt werden.
</Note>

## GitHub-Zweigwechsel

Durch Aktivieren von GitHub Branch Switching können Sie den Branch auswählen, der von ZenOpus bearbeitet wird. Nach der Aktivierung können Sie den Zweig wechseln, indem Sie die folgenden Schritte ausführen.

1. Gehen Sie zu den **Einstellungen** eines Projekts
1. Gehen Sie zur Registerkarte **GitHub**
1. Klicken Sie auf das Dropdown-Menü „Zweig“ und wählen Sie den Zweig aus, zu dem Sie wechseln möchten

<Note>
ZenOpus synchronisiert standardmäßig mit dem „Hauptzweig“.
</Note>
