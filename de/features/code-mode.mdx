---
title: "Projektcode anzeigen und bearbeiten"
description: "Zeigen Sie den Quellcode Ihres Projekts direkt im ZenOpus-Code-Editor an und bearbeiten Sie ihn manuell."
---
Mit dieser Funktion erhalten Sie direkten Zugriff auf die zugrunde liegende Codebasis Ihres Projekts. Sie können Dateien prüfen, gezielte Änderungen vornehmen und Probleme beheben, ohne das Tool wechseln oder Ihr Projekt exportieren zu müssen.

![Build-Code-Editor](/images/zenopus/features/code-mode/build-code-editor.png)

### Was Sie tun können

In der Code-Editor-Ansicht können Sie:

- Durchsuchen Sie Ihre gesamte Projektdateistruktur
- Durchsuchen Sie Dateien und springen Sie zu bestimmten Komponenten
- Dateien direkt prüfen und bearbeiten
- Code formatieren
- Dateiinhalt kopieren
- Dateien herunterladen
- Laden Sie die vollständige Projektcodebasis als ZIP herunter
- Referenzieren Sie Dateien im Chat, um gezielte Bearbeitungen und Diskussionen schneller und klarer zu gestalten

<Tip>
**Codedateien im Chat erwähnen**.   

 Geben Sie `@` ein oder klicken Sie im Code-Editor auf die Schaltfläche **Referenzdatei im Chat**, um bestimmte Dateien zu erwähnen. Zum Beispiel:   

`@src/components/UserProfile.tsx add a loading state here`
</Tip>

### Gut zu wissen

- Hier vorgenommene Änderungen wirken sich sofort auf Ihr Projekt aus
- Die Codebearbeitung ist nur bei kostenpflichtigen Plänen verfügbar
- Dies ist für manuelle Anpassungen gedacht und beim Einbau in Agent mode nicht erforderlich.
