Die **ZenOpus-API** ermöglicht das programmgesteuerte Erstellen und Teilen von ZenOpus-Apps – und verwandelt Ideen, Eingabeaufforderungen oder sogar Bilder mit einem einzigen Link in vollständig generierte Anwendungen.

Es ist für Entwickler, Ersteller und Teams konzipiert, die von überall im Web aus **ZenOpus in ihre Arbeitsabläufe integrieren** oder **die App-Generierung automatisieren** möchten.

## Was ist die ZenOpus-API?

Die ZenOpus-API ist eine wachsende Suite von Funktionen, die die Möglichkeiten von ZenOpus über die Weboberfläche hinaus erweitern.

Mit der ersten Version, [**Build with URL**](/de/integrations/build-with-url), können Sie Apps direkt aus einer gemeinsam nutzbaren URL generieren.

Damit können Sie:

- Betten Sie **Build with ZenOpus**-Schaltflächen in Ihre Website oder App ein.
- Teilen Sie App-generierende Links mit Teamkollegen oder Kunden.
- Integrieren Sie die ZenOpus-App-Erstellung in interne Tools oder Skripte.
- Erstellen Sie sofort Prototypen über einen einzigen Link.

## Warum die ZenOpus-API verwenden?

- **Automatisierung:** Lösen Sie die App-Erstellung aus Code, Dashboards oder Workflows aus.
- **Integration:** Integrieren Sie ZenOpus in Ihr eigenes Produkterlebnis.
- **Geschwindigkeit:** Generieren und teilen Sie neue App-Ideen schneller als je zuvor.
- **Teilbarkeit:** URLs machen die Zusammenarbeit einfach und reibungslos.
