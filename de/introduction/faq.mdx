## Erste Schritte

<AccordionGroup>
<Accordion title="Was ist ZenOpus?">
ZenOpus ist eine KI-gestützte Plattform, die es Benutzern aller Erfahrungsstufen ermöglicht, Full-Stack-Websites in natürlicher Sprache zu erstellen. Beschreiben Sie einfach, was Sie wollen, und ZenOpus erstellt es für Sie.
</Accordion>
<Accordion title="Was kann ich mit ZenOpus bauen?">
Wenn Sie es sich vorstellen können, können Sie es mit ZenOpus erstellen. Von einfachen Landingpages bis hin zu leistungsstarken Tools, die auf Nischenanwendungsfälle zugeschnitten sind: ZenOpus bietet Ihnen die Flexibilität, ohne Einschränkungen alles zu erstellen, was Sie benötigen.
</Accordion>
<Accordion title="Benötige ich Programmiererfahrung, um mit ZenOpus zu erstellen?">
Nein. ZenOpus ist für alle Erfahrungsstufen konzipiert. Unabhängig davon, ob Sie Entwickler oder technisch nicht versierter Benutzer sind, können Sie Websites mit ZenOpus erstellen.
</Accordion>
<Accordion title="Wie erstelle ich ein Projekt?">
Um ein Projekt zu erstellen, geben Sie einfach eine Nachricht in das Nachrichtenfeld im Dashboard ein. Beschreiben Sie einfach, was Sie erstellen möchten, und ZenOpus erstellt daraus einen Ausgangspunkt für Ihr Projekt. Denken Sie daran, klein anzufangen und schrittweise aufzubauen!
</Accordion>
<Accordion title="Kann ich ein bestehendes Projekt wiederverwenden?">
Ja, Sie können ein bestehendes öffentliches Projekt oder eines Ihrer eigenen remixen. Durch Remixen können Sie den aktuellen Status eines Projekts als Ausgangspunkt wiederverwenden und darauf aufbauen. Dies ist eine großartige Möglichkeit, neue Ideen zu erkunden, Anpassungen vorzunehmen oder mit verschiedenen Änderungen zu iterieren und gleichzeitig die Originalversion beizubehalten.
</Accordion>
</AccordionGroup>

## Bauen mit ZenOpus

<AccordionGroup>
<Accordion title="Wie baue ich effizient mit ZenOpus?">
Beim effizienten Bauen mit ZenOpus kommt es vor allem auf die Verwendung der richtigen Techniken an. Mit etwas Übung kann es jeder schaffen. Um Ihnen den Einstieg zu erleichtern, lesen Sie unseren [Leitfaden](https://mastering-zenopus.zenopus.app/).
</Accordion>
<Accordion title="Wie sollte ich meine Eingabeaufforderungen strukturieren, um die besten Ergebnisse zu erzielen?">
- Seien Sie klar und konkret.
- Konzentrieren Sie sich jeweils auf eine Funktion.
- Bauen Sie schrittweise auf.
- Verwenden Sie Schlagworte, um das Design richtig hinzubekommen.
</Accordion>
<Accordion title="Was soll ich tun, wenn ein Fehler auftritt?">
<Steps>
<Step title="Schaltfläche „Reparaturversuch“.">
Verwenden Sie die Option **Reparaturversuch**. ZenOpus wird versuchen, das Problem automatisch zu lösen. Durch die Nutzung der Schaltfläche werden keine Credits abgezogen.
</Step>
<Step title="Untersuchen">
Wenn **Try-to-fix** das Problem nicht löst, bitten Sie den ZenOpus-Agenten, das Problem zu untersuchen. Erklären Sie, was Sie sehen und welche spezifischen Teile nicht funktionieren.
</Step>
<Step title="Zu einer älteren Version zurückkehren">
Wenn das nicht funktioniert, können Sie eine Vorschau einer älteren Arbeitsversion anzeigen und dann bei Bedarf zu dieser Version zurückkehren oder eine frühere Nachricht wiederherstellen und bearbeiten, um einen anderen Ansatz zu erkunden. Nichts geht verloren – alle nach diesem Zeitpunkt vorgenommenen Änderungen bleiben im Chat und können jederzeit erneut angewendet werden.
</Step>
</Steps>
</Accordion>
<Accordion title="Was ist Refactoring und warum ist es wichtig?">
Refactoring verbessert die Codestruktur und -effizienz, ohne die Funktionalität zu verändern. Es hilft:

- Optimieren Sie die Leistung, indem Sie Redundanzen beseitigen.
- Verbessern Sie die Wartbarkeit für zukünftige Updates.
- Vermeiden Sie Konflikte, indem Sie die Logik klarer strukturieren.
</Accordion>
<Accordion title="Kann ich mein Projekt mit einem Backend verbinden, um Daten zu speichern?">
Wenn Sie Ihrer Anwendung Backend-Funktionen hinzufügen müssen, können Sie:

- Stellen Sie eine Verbindung zum integrierten Backend von ZenOpus her –  [ZenOpus Cloud](/de/features/cloud)
- Verwenden Sie die native [Supabase-Integration](/de/integrations/supabase) von ZenOpus.
</Accordion>
<Accordion title="Kann ich meiner Website ein Login hinzufügen?">
Ja, Sie können die Authentifizierung hinzufügen, indem Sie [ZenOpus Cloud](/de/features/cloud) verwenden oder mit [Supabase](/de/integrations/supabase) integrieren. Fordern Sie einfach die KI auf, es hinzuzufügen, und befolgen Sie die Anweisungen.
</Accordion>
<Accordion title="Kann ich meinem Projekt APIs von Drittanbietern hinzufügen?">
Ja, Sie können APIs wie OpenAI, Stripe und mehr integrieren, indem Sie sie in Ihrer Eingabeaufforderung angeben. Lesen Sie mehr [hier](/de/integrations/introduction).
</Accordion>
<Accordion title="Kann ich vertrauliche API-Schlüssel in ZenOpus speichern?">
Nein! Geben Sie Ihre API-Schlüssel niemals direkt in ZenOpus ein. Wenn Sie eine API integrieren, empfehlen wir Ihnen, Geheimnisse in [ZenOpus Cloud](/de/features/cloud) oder [Supabase](/de/integrations/supabase) zu verwenden, um Ihre API-Schlüssel in Kombination mit ihren Edge-Funktionen zu speichern.
</Accordion>
<Accordion title="Wie füge ich Zahlungen zu meiner Website hinzu?">
Sie können Zahlungen hinzufügen, indem Sie Stripe zu Ihrem Projekt hinzufügen. Informationen zur Einrichtung finden Sie unter [Stripe-Integration](/de/integrations/stripe).
</Accordion>
<Accordion title="Wie kann ich SEO für mein Projekt verbessern?">
Sorgen Sie für die besten Ergebnisse, indem Sie unserem Leitfaden [Best Practices für SEO und GEO implementieren](/de/tips-tricks/seo-geo) folgen.
</Accordion>
<Accordion title="Wie lange dauert die Fertigstellung eines Projekts?">
Es kommt auf die Komplexität an. Indem Sie Aufgaben in kleinere Schritte unterteilen, vorausplanen und auf der Grundlage von Feedback verfeinern, können Sie schnell vorankommen und gleichzeitig die Qualität sicherstellen.
</Accordion>
</AccordionGroup>

## Funktionen

<AccordionGroup>
<Accordion title="Wie finde ich die Projekteinstellungen?">

Ja. Wenn Sie zu einem privaten Projekt navigieren, können Sie beim Projektinhaber den Zugriff als Betrachter oder Redakteur anfordern. Dies schützt private Arbeiten und erleichtert gleichzeitig die Zusammenarbeit.
</Accordion>
<Accordion title="Kann ich das ZenOpus-Abzeichen ausblenden?">
Ja, navigieren Sie zu den **Projekteinstellungen**, wo Sie die Option zum Ausblenden des Abzeichens finden. Nur zahlende Benutzer können das Abzeichen deaktivieren.
</Accordion>
<Accordion title="Kann ich ein Projekt umbenennen?">
Ja, navigieren Sie zu den **Projekteinstellungen** und dort finden Sie die Option zum Umbenennen des Projekts.
</Accordion>
<Accordion title="Wie kopiere (remixe) ich ein Projekt?">
Um eine Kopie eines Projekts zu erstellen, **remixen** Sie es. Gehen Sie zu den **Projekteinstellungen** und wählen Sie die Option **Remix**.

Durch das Remixen wird eine exakte Kopie des Projekts in Ihrem aktuell ausgewählten Workspace erstellt. Anschließend können Sie es selbstständig bearbeiten.

<Note>
Sie können öffentliche Projekte von anderen Benutzern nur dann remixen, wenn das Projekt nicht mit Supabase verbunden ist.
</Note>
</Accordion>
<Accordion title="Kann ich ein Projekt löschen?">
Ja, navigieren Sie zu den **Projekteinstellungen** und dort finden Sie die Option zum **Löschen** des Projekts.

<Note>
Dies ist endgültig und es gibt keine Möglichkeit, gelöschte Projekte wiederherzustellen.
</Note>
</Accordion>

Plan mode (früher Chat-Modus) eignet sich ideal zum Planen von Funktionen, zum Debuggen von Problemen und zum Verstehen von Änderungen, bevor Code geschrieben oder geändert wird. Weitere Informationen finden Sie unter [Brainstorming mit Plan mode](/de/features/plan-mode).

## ZenOpus-API

<Accordion title="Was kann ich mit der ZenOpus-API tun?">
Die **ZenOpus-API** ermöglicht das programmgesteuerte Erstellen und Teilen von ZenOpus-Apps – und verwandelt Ideen, Eingabeaufforderungen oder sogar Bilder mit einem einzigen Link in vollständig generierte Anwendungen.

Mit der ersten Version, **Build with URL**, können Sie Apps direkt aus einer gemeinsam nutzbaren URL generieren. Damit können Sie:

- Betten Sie eine Schaltfläche **Build with ZenOpus** in Ihre Website oder Anwendung ein.
- Teilen Sie vordefinierte Eingabeaufforderungen über URLs mit Freunden, Kollegen oder in sozialen Medien.
- Automatisieren Sie die App-Generierung als Teil interner Arbeitsabläufe.
- Erstellen Sie mithilfe von URLs mit Eingabeaufforderungen und Referenzbildern schnell Prototypen für Apps.

Erfahren Sie mehr über die Funktion und reale Anwendungsfälle in unserem Ankündigungsbeitrag: [Apps über einen Link erstellen mit ZenOpus Mit URL erstellen](https://zenopus.dev/blog/introducing-zenopus-api-build-with-url).

Lesen Sie den vollständigen technischen Leitfaden hier: [ZenOpus API: Build with URL-Dokumentation](/de/integrations/build-with-url).
</Accordion>

## Verwalten Sie Ihr Konto

<AccordionGroup>
<Accordion title="Kann ich die E-Mail-Adresse ändern, mit der ich mich anmelde?">
Eine direkte Änderung der E-Mail-Adresse ist derzeit leider nicht möglich. Um dieses Problem zu umgehen, können Sie ein neues Konto mit Ihrer gewünschten E-Mail-Adresse erstellen und Ihre Projekte übertragen, indem Sie die folgenden Schritte ausführen:

1. Machen Sie Ihre bestehenden Projekte vorübergehend öffentlich und kopieren Sie die URL.
1. Greifen Sie auf die URL zu, während Sie bei Ihrem neuen Konto angemeldet sind.
1. Mischen Sie die Projekte erneut in Ihr neues Konto. Bitte beachten Sie, dass mit Supabase verbundene Projekte von Supabase getrennt werden müssen, bevor Sie sie remixen können.
1. Sobald die Projekte übertragen wurden, können Sie sie bei Bedarf wieder privat machen.

Hinweis: Wenn Ihr Ziel Workspace einen kostenpflichtigen Plan hat, ist möglicherweise das Remixen öffentlicher Projekte für den Zugriff erforderlich. Remix-Projekte übernehmen keine Funktionen auf Abonnementebene, es sei denn, das neue Workspace verfügt auch über diese.
</Accordion>
<Accordion title="Kann ich mein Konto löschen?">
Um Ihr Konto zu löschen, gehen Sie zu **Einstellungen → Ihr Konto**, scrollen Sie nach unten und klicken Sie auf **Konto löschen**. Dadurch werden Ihr Konto und alle damit verbundenen Daten dauerhaft entfernt.
</Accordion>
<Accordion title="Kann ich ein gelöschtes Konto reaktivieren?">
Nein, Sie können ein gelöschtes Konto nicht reaktivieren. Nachdem Sie jedoch auf die Schaltfläche zum Löschen des Kontos geklickt haben, kann diese Aktion innerhalb von 48 Stunden abgebrochen werden, um die Löschung zu stoppen.
</Accordion>
<Accordion title="Kann ich den Workspace-Besitzer ändern?">
Ja, ein Eigentümer kann mehrere Workspace-Eigentümer hinzufügen.
</Accordion>
<Accordion title="Wie kann ich die Projektverantwortung auf eine andere Person übertragen?">
Eine Möglichkeit, dies zu umgehen, besteht darin, das Projekt neu zu mischen. Die Person, die das Projekt neu mischt, ist nun Eigentümer dieser Version.
</Accordion>
<Accordion title="Wie aktiviere ich die Zwei-Faktor-Authentifizierung (2FA)?">
**Zwei-Faktor-Authentifizierung (2FA)** fügt Ihrem ZenOpus-Konto eine zusätzliche Sicherheitsebene hinzu, indem bei der Anmeldung zusätzlich zu Ihrer üblichen Anmeldemethode ein Einmalcode erforderlich ist.

### Richten Sie 2FA ein

1. Gehen Sie zu **Einstellungen → Konto → Ihr Konto** und aktivieren Sie **Zwei-Faktor-Authentifizierung**.
1. Wählen Sie eine Verifizierungsmethode aus und richten Sie sie ein, indem Sie den Anweisungen auf dem Bildschirm folgen:
    - **Authenticator-App (empfohlen)**  
    Verwenden Sie eine App wie Google Authenticator, Microsoft Authenticator, Authy oder 1Password, um zeitbasierte Einmalcodes zu generieren.
    - **Telefon (SMS)**  
    Erhalten Sie 6-stellige Bestätigungscodes per SMS.
1. Klicken Sie auf **Überprüfen und aktivieren**.

#### Anmeldung mit 2FA

Wenn Sie sich anmelden:

1. Vervollständigen Sie Ihre übliche Anmeldemethode.
1. Wählen Sie eine Verifizierungsmethode.
1. Geben Sie den 6-stelligen Code aus Ihrer Authentifizierungs-App oder SMS ein.

#### Verwaltung von 2FA

- Sie können 2FA-Methoden jederzeit zu Ihren Kontoeinstellungen hinzufügen oder entfernen.
- Wir empfehlen, **mindestens zwei Methoden** konfiguriert zu lassen, um eine Sperrung zu vermeiden.
</Accordion>
<Accordion title="Was sind Vibe-Codierungsstufen und wie füge ich sie zu meinem LinkedIn-Konto hinzu?">
Vibe-Codierungsstufen spiegeln wider, wie aktiv Sie ZenOpus nutzen, basierend auf der Anzahl der Nachrichten, die Sie beim Erstellen senden. Ihr Level erhöht sich automatisch, wenn Sie ZenOpus mehr nutzen.

Die Level sind:

- **Bronze**: Erste Schritte mit ZenOpus
- **Silber**: Regulärer ZenOpus-Benutzer
- **Gold**: Aktiver ZenOpus-Builder
- **Platinum**: Sehr engagierter ZenOpus-Builder
- **Diamant**: Erstklassiges ZenOpus-Engagement

Sie können Ihr Level unter **Einstellungen → Ihr Konto** einsehen. Optional können Sie Ihr LinkedIn-Konto verbinden, um unter **Lizenzen & Zertifizierungen** Ihr aktuelles Level anzuzeigen. Dies ist völlig optional und kann jederzeit getrennt werden.

Weitere Informationen finden Sie unter [Vibe-Coding-LinkedIn-Zertifizierung](/de/tips-tricks/linkedin-certification) (Beta).
</Accordion>
</AccordionGroup>
