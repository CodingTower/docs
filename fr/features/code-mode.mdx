---
title: "Afficher et modifier le code du projet"
description: "Affichez et modifiez manuellement le code source de votre projet directement dans l'éditeur de code ZenOpus."
---
Cette fonctionnalité vous donne un accès direct à la base de code sous-jacente de votre projet. Vous pouvez inspecter les fichiers, effectuer des modifications ciblées et résoudre les problèmes sans changer d'outil ni exporter votre projet.

![Éditeur de code de construction](/images/zenopus/features/code-mode/build-code-editor.png)

### Ce que tu peux faire

Depuis la vue de l'éditeur de code, vous pouvez :

- Parcourez la structure complète de vos fichiers de projet
- Rechercher dans les fichiers et accéder à des composants spécifiques
- Inspecter et modifier les fichiers directement
- Formater le code
- Copier le contenu du fichier
- Télécharger des fichiers
- Téléchargez la base de code complète du projet au format ZIP
- Référencez des fichiers dans le chat pour rendre les modifications et les discussions ciblées plus rapides et plus claires

<Tip>
**Mentionnez les fichiers de code dans le chat**.   

 Tapez `@`ou cliquez sur le bouton **fichier de référence dans le chat** dans l'éditeur de code pour mentionner des fichiers spécifiques. Par exemple :   

`@src/components/UserProfile.tsx add a loading state here`
</Tip>

### Bon à savoir

- Les modifications apportées ici affectent immédiatement votre projet
- La modification du code est disponible uniquement sur les forfaits payants
- Ceci est destiné aux ajustements manuels et n'est pas requis lors de la construction en Agent mode
