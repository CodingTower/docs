## Que sont les modèles de conception ?

Les modèles de conception sont disponibles sur les forfaits **Business** et **Enterprise**. Ils permettent aux équipes de réutiliser les conceptions et les structures de projet approuvées pour maintenir un style, une image de marque et une expérience utilisateur cohérents dans tous les projets.

Un modèle de conception est un projet ZenOpus qui a été marqué comme réutilisable. Lorsqu'un modèle est utilisé, l'intégralité de la base de code du projet est copiée dans le nouveau projet, y compris :

- Structure du fichier
- Composants et styles
- Fichiers de configuration
- Montage du projet et échafaudage

Les projets créés à partir de modèles de conception sont totalement indépendants et peuvent être modifiés librement.

## Pourquoi utiliser des modèles de conception ?

Utilisez des modèles de conception pour :

- Maintenir des systèmes de conception et une architecture cohérents
- Démarrez de nouveaux projets à partir de bases de code éprouvées
- Réduisez les travaux de configuration répétitifs
- Partagez les meilleures pratiques sur un Workspace

Les modèles de conception aident les équipes à avancer plus rapidement tout en gardant les projets alignés.

## Conditions préalables

Pour utiliser des modèles de conception :

- Vous devez disposer d'un forfait **Business ou Entreprise**
- Le projet modèle doit avoir une visibilité **publique ou Workspace**
- Vous devez être éditeur, administrateur ou propriétaire de projet pour définir des projets comme modèles
- Vous devez être un administrateur ou un propriétaire Workspace pour gérer les modèles par défaut

## Configurer un modèle de conception

Pour configurer un modèle de conception, vous devez être éditeur, administrateur ou propriétaire de projet.

1. Ouvrez un projet existant.
1. Accédez à **Paramètres du projet.**
1. Définissez la visibilité du projet sur **public** ou **Workspace.**
1. Activez **Utiliser comme modèle.**

Le projet apparaît dans la section **Workspace settings → Templates** et peut être utilisé lors de la création de nouveaux projets.

## Démarrer un projet à partir d'un modèle de conception

Pour utiliser un modèle lors de la création d'un nouveau projet :

1. Cliquez sur `+` dans la boîte de dialogue.
1. Accédez à **Conception → Utiliser un modèle**.
1. Choisissez un modèle et créez votre projet comme d'habitude.

Le modèle sélectionné est appliqué comme base du projet. Votre invite contrôle toujours ce qui est généré.

## Modèles par défaut

Chaque Workspace peut avoir **un modèle de conception par défaut**. Les modèles par défaut aident à standardiser la façon dont les projets sont créés sur un Workspace. Vous devez être un administrateur ou un propriétaire Workspace pour gérer les modèles par défaut.

Pour définir un modèle par défaut :

1. Accédez aux paramètres **Workspace → Modèles**.
1. Passez la souris sur un modèle et sélectionnez **Définir par défaut**.

Lorsqu'un modèle par défaut est défini, il apparaît dans la zone d'invite et est appliqué automatiquement aux nouveaux projets. Vous pouvez remplacer la sélection à tout moment.

Pour supprimer un modèle par défaut :

1. Accédez aux paramètres **Workspace → Modèles**.
1. Passez la souris sur le modèle et supprimez la valeur par défaut.
