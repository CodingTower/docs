You can enable and disable different Labs-features by going to your [Account Settings](https://zenopus.dev/settings/account), and then navigating to **Labs**.

<Note>
These experimental features may change or be removed at any time.
</Note>

## GitHub branch switching

By enabling GitHub Branch Switching you can select the branch that ZenOpus edits. Once enabled you can switch branch by following these steps.

1. Go to the **Settings** of a project
1. Go to **GitHub** tab
1. Click the branch dropdown, and choose the branch you want to switch to

<Note>
ZenOpus defaults to syncing with the “main” branch
</Note>
