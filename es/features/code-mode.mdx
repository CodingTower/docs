---
title: "Ver y editar el código del proyecto"
description: "Vea y edite manualmente el código fuente de su proyecto directamente en el editor de código ZenOpus."
---
Esta función le brinda acceso directo al código base subyacente de su proyecto. Puede inspeccionar archivos, realizar ediciones específicas y solucionar problemas sin cambiar de herramienta ni exportar su proyecto.

![Editor de código de compilación](/images/zenopus/features/code-mode/build-code-editor.png)

### que puedes hacer

Desde la vista del editor de código, puede:

- Explore la estructura completa de archivos de su proyecto
- Busque en archivos y salte a componentes específicos
- Inspeccionar y editar archivos directamente
- Código de formato
- Copiar el contenido del archivo
- Descargar archivos
- Descargue el código base completo del proyecto como ZIP
- Archivos de referencia en el chat para hacer que las ediciones y debates específicos sean más rápidos y claros

<Tip>
**Mencionar archivos de código en el chat**.   

 Escriba `@` o haga clic en el botón **archivo de referencia en el chat** en el editor de código para mencionar archivos específicos. Por ejemplo:   

__CÓDIGO0__
</Tip>

### Es bueno saberlo

- Los cambios realizados aquí afectan inmediatamente a su proyecto.
- La edición de código está disponible solo en planes pagos
- Esto está pensado para ajustes manuales y no es necesario al construir en Agent mode
