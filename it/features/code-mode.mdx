---
title: "Visualizza e modifica il codice del progetto"
description: "Visualizza e modifica manualmente il codice sorgente del tuo progetto direttamente nell'editor di codice ZenOpus."
---
Questa funzionalità ti dà accesso diretto alla base di codice sottostante del tuo progetto. Puoi ispezionare file, apportare modifiche mirate e risolvere problemi senza cambiare strumento o esportare il tuo progetto.

![Crea editor di codice](/images/zenopus/features/code-mode/build-code-editor.png)

### Cosa puoi fare

Dalla visualizzazione dell'editor di codice è possibile:

- Sfoglia la struttura completa dei file di progetto
- Cerca tra i file e passa a componenti specifici
- Ispeziona e modifica direttamente i file
- Codice formato
- Copia il contenuto del file
- Scarica file
- Scarica la base di codice completa del progetto come ZIP
- File di riferimento in chat per apportare modifiche e discussioni mirate in modo più rapido e chiaro

<Tip>
**Menziona i file di codice nella chat**.   

 Digita `@`o fai clic sul pulsante **file di riferimento nella chat** nell'editor del codice per menzionare file specifici. Ad esempio:   

__CODICE0__
</Tip>

### Buono a sapersi

- Le modifiche apportate qui avranno effetto immediato sul tuo progetto
- La modifica del codice è disponibile solo sui piani a pagamento
- Questo è destinato alle regolazioni manuali e non è necessario durante la creazione in Agent mode
