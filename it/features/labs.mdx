Puoi attivare e disattivare diverse funzionalità di Labs accedendo alle [Impostazioni account](https://zenopus.dev/settings/account) e quindi accedendo a **Labs**.

<Note>
Queste funzionalità sperimentali possono cambiare o essere rimosse in qualsiasi momento.
</Note>

## Cambio di ramo GitHub

Abilitando GitHub Branch Switching è possibile selezionare il ramo che ZenOpus modifica. Una volta abilitato, puoi cambiare ramo seguendo questi passaggi.

1. Vai alle **Impostazioni** di un progetto
1. Vai alla scheda **GitHub**
1. Fai clic sul menu a discesa del ramo e scegli il ramo a cui desideri passare

<Note>
ZenOpus per impostazione predefinita è sincronizzato con il ramo "principale".
</Note>
