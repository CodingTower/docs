---
title: "プロジェクトコードの表示と編集"
description: "ZenOpus コード エディターでプロジェクトのソース コードを直接表示し、手動で編集します。"
---
この機能により、プロジェクトの基礎となるコードベースに直接アクセスできるようになります。ツールを切り替えたりプロジェクトをエクスポートしたりせずに、ファイルを検査し、対象を絞った編集を行って、問題のトラブルシューティングを行うことができます。

![ビルド コード エディター](/images/zenopus/features/code-mode/build-code-editor.png)

### できること

コード エディター ビューからは、次のことができます。

- 完全なプロジェクト ファイル構造を参照する
- ファイル全体を検索し、特定のコンポーネントにジャンプします
- ファイルを直接検査して編集する
- フォーマットコード
- ファイルの内容をコピーする
- ファイルをダウンロードする
- 完全なプロジェクトのコードベースを ZIP 形式でダウンロードする
- チャットでファイルを参照して、対象を絞った編集やディスカッションをより迅速かつ明確に行うことができます。

<Tip>
**チャットでコード ファイルについて言及します**。   

 `@` と入力するか、コード エディターで **チャット内の参照ファイル** ボタンをクリックして、特定のファイルを言及します。たとえば:   

__コード0__
</Tip>

### 知っておいてよかった

- ここで加えた変更はすぐにプロジェクトに影響します
- コード編集は有料プランでのみ利用可能です
- これは手動調整を目的としており、Agent mode でビルドする場合には必要ありません。
