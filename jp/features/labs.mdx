[アカウント設定](https://zenopus.dev/settings/account) に移動し、**Labs** に移動して、さまざまな Labs 機能を有効または無効にできます。

<Note>
これらの実験的な機能はいつでも変更または削除される可能性があります。
</Note>

## GitHub ブランチ切り替え

GitHub ブランチ切り替えを有効にすることで、ZenOpus が編集するブランチを選択できます。有効にすると、次の手順に従ってブランチを切り替えることができます。

1. プロジェクトの**設定**に移動します
1. **GitHub** タブに移動します
1. ブランチのドロップダウンをクリックし、切り替え先のブランチを選択します

<Note>
ZenOpus はデフォルトで「メイン」ブランチと同期します
</Note>
