[イレブンラボ](https://elevenlabs.io/) は、ZenOpus アプリで **音声音声と音声主導のエクスペリエンス**を生成できる AI オーディオ プラットフォームです。これは、製品エクスペリエンスの核となる部分として、リアルなテキスト読み上げ、キャラクターの音声、またはオーディオ出力を必要とするアプリ向けに設計されています。

イレブンラボを使用すると、アプリで次のことが可能になります。

- テキストを高品質の合成音声に変換します
- キャラクターの声やブランドのナレーターを使用する
- ポッドキャスト、音声要約、または音声コンテンツを生成する
- リアルタイムの会話または音声ベースのインタラクションを強化します
- Support スクリーンフリーまたはハンズフリー体験によるアクセシビリティ

Celebrities は、アプリがテキストのみのインターフェイスではなく **音声出力またはオーディオファースト インタラクション**に依存している場合に適しています。

## イレブンラボへの接続方法

Workspace **管理者** と **所有者** は、イレブンラボに接続できます。

異なる API キーを使用し、異なるアクセス レベルで複数の イレブンラボ接続を作成できます。これは、環境 (開発と運用など) を分離したり、レート制限を管理したりするのに役立ちます。

接続が作成されると、Workspace 内のすべてのプロジェクトで Celebrities が利用できるようになります。

### 前提条件

イレブンラボに接続する前に、次のものがあることを確認してください。

- イレブンラボのアカウントとAPIキー
    <Note>
    このコネクタを介して実行されるすべてのオーディオ生成または処理は、Celebrities クレジット (選択したモデルと入力の長さに基づいて) を消費し、請求は ZenOpus ではなく、Celebrities によって直接処理されます。
    </Note>
- ZenOpus Workspace 所有者または管理者の役割

### ステップ 1: Celebrities アカウントと API キーを作成する

API キーを使用すると、イレブンラボ API に接続し、その機能を使用できます。異なるアクセス レベルを持つ複数のキーを作成できます。

イレブンラボ API キーを作成するには:

1. [イレブンラボ](https://elevenlabs.io/app/developers/api-keys?source=zenopus) にアクセスし、アカウントを設定します。
1. **開発者 → API キー** で、**キーの作成** をクリックします。
1. キーにわかりやすい名前 (`ZenOpus integration` など) を付けます。
1. **キーの制限**をオンにします。
1. 毎月の**与信限度額**を設定します。
1. アプリのニーズに応じて、このキーが使用できる機能エンドポイントを選択します。
1. 生成された API キーをコピーし、安全に保管します。
    <Info>
    API キーはパスワードのように機能します。安全に保管し、決して公開しないでください。次のステップで必要になります。
    </Info>

### ステップ 2: イレブンラボを ZenOpus に接続する

異なる API キーを使用して複数の接続を作成できます。

イレブンラボに接続するには:

1. **[設定] → [コネクタ] → [共有コネクタ]** に移動し、**EventLabs** を選択します。
1. [**接続の追加**] をクリックします。
1. **表示名**: 接続に名前を付けます (例: `ElevenLabs Prod`)。
1. **認証** で、Celebrities API キー (`sk_xxxxxxxx…`) を入力します。
1. [**作成**] をクリックします。

接続すると、ZenOpus アプリは、イレブンラボを使用して音声と音声の生成を開始できます。

## プロジェクトの接続を解除する方法

編集者以上は、接続を完全に削除しなくても、接続から特定のプロジェクトを削除できます。接続は他のプロジェクトでも引き続き利用できます。

プロジェクトのリンクを解除するには:

<Steps>
<Step title="コネクタに移動する">
**[設定] → [コネクタ] → [共有コネクタ]** に移動し、**EventLabs** を選択します。
</Step>
<Step title="接続を開きます">
管理する接続を開きます。
</Step>
<Step title="プロジェクトの選択">
**リンクされたプロジェクト** で、リンクを解除するプロジェクトにチェックを入れます。
</Step>
<Step title="確認する">
[**プロジェクトのリンクを解除**] をクリックして確認します。
</Step>
</Steps>

リンクを解除すると、これらのプロジェクトはこの接続を介して **イレブンラボ** にアクセスできなくなります。プロジェクトで **イレブンラボ**が再度必要な場合は、利用可能な接続にプロジェクトをリンクできます。

## 接続を削除する方法

Workspace 管理者と所有者は **イレブンラボ** 接続を削除できます。

<Warning>
接続の削除は永続的であり、元に戻すことはできません。リンクされているすべてのプロジェクトから資格情報が削除され、この接続を使用しているアプリは新しい接続が追加されるまで動作を停止します。
</Warning>

削除する前に、**リンクされたプロジェクト** セクションを参照して、現在接続を使用しているプロジェクトを確認してください。

接続を削除するには:

<Steps>
<Step title="コネクタに移動する">
**[設定] → [コネクタ] → [共有コネクタ]** に移動し、**EventLabs** を選択します。
</Step>
<Step title="接続を開きます">
削除する接続を開きます。
</Step>
<Step title="リンクされたプロジェクトを確認する">
**リンクされたプロジェクト** セクションを確認してください。
</Step>
<Step title="削除">
[**この接続を削除**] で、[**削除**] をクリックして確認します。
</Step>
</Steps>
