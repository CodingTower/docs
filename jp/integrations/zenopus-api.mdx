**ZenOpus API** を使用すると、ZenOpus アプリをプログラムで作成および共有できるようになり、アイデア、プロンプト、画像さえも 1 つのリンクで完全に生成されたアプリケーションに変換できます。

**ZenOpus をワークフローに統合**、または Web 上のどこからでも **アプリ生成を自動化**したい開発者、クリエイター、チーム向けに設計されています。

## ZenOpus API とは何ですか?

ZenOpus API は、Web インターフェイスを超えて ZenOpus でできることを拡張する、成長を続ける機能スイートです。

最初のリリースである [**Build with URL**](/jp/integrations/build-with-url) では、共有可能な URL から直接アプリを生成できます。

これを使用すると、次のことが可能になります。

- **ZenOpus** ボタンをサイトまたはアプリに埋め込みます。
- アプリ生成リンクをチームメイトや顧客と共有します。
- ZenOpus アプリの作成を内部ツールまたはスクリプトに統合します。
- 単一のリンクから即座にプロトタイプを作成します。

## ZenOpus API を使用する理由は何ですか?

- **自動化:** コード、ダッシュボード、またはワークフローからアプリの作成をトリガーします。
- **統合:** ZenOpus を独自の製品エクスペリエンスに組み込みます。
- **スピード:** 新しいアプリのアイデアをこれまでより速く生成して共有します。
- **共有性:** URL により、コラボレーションが簡単かつスムーズになります。

## ZenOpus API の次のステップは何ですか?

これはほんの始まりにすぎません。 ZenOpus API は、より多くのエンドポイントと統合オプションで拡張され続けるため、ZenOpus を開発および自動化のワークフローにどのように適合させるかをより詳細に制御できるようになります。
